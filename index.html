Welcome to the COOKIN Style Guide

Here you will find common elements on the Tamooki website to help build and create a more consistent and concise site

What does the company do, or offer?

COOKIN is a restaurant which seats people on specially made tables featuring a barbeque grill. Customers buy prepped meats and vegetables to cook using the grills on their table. Customers can make reservations for tables. People also have the option to buy the prepped food to take home. It's exactly like kbbq but with American food such as burgers, wings, etc instead

Who is the company to potential customers?

The brand's persona will be somewhat informal but still professional so it is suitable for most audiences. No overly casual or country jargon/speak. Stay proffesional but accessible to retain as many users as possible.

Why might this company entice someone to purchase a product or service?

This restaurant would bring people in for a unique eating experience, good food and a great time to spend with people. Also gives people the barbeque experience for people who cannot have a barbeque in their own home.

Interactive Elements

In-text Link

Sample code:

<a href="#">This is a link</a>

Rendered element:

This is a link

Navigation Link

A Navigation Link is a button that when clicked will navigate to another page

By default, each button element will be a small button if no size is assigned. Button sizes are split into small, medium, and large and can be assigned by using the following CSS selectors:

  • .button-sm -> Small Button
  • .button-md -> Medium Button
  • .button-lg -> Large Button

CSS selector:

.button-nav

Sample code:

<a class="button-nav button-sm" href="#">Default Button</a>
<a class="button-nav button-md" href="#">Default Button</a>
<a class="button-nav button-lg" href="#">Default Button</a>

Rendered element:

Default Button Default Button Default Button

Submit/Reset Buttons

A Submit/Reset Button is a button that when clicked will perform an action. They are a special type of Navigation Button thus they also use the same sizing stylings

When hovered, these functional buttons do not have any animations when hovered to differentiate from Navigation Link buttons

CSS selector:

.submit-button .reset-button

CSS combined element selector:

.button-nav

Sample code:

<a class="button-nav button-md submit-button" href="#">Submit Button</a>
<a class="button-nav button-md reset-button" href="#">Reset Button</a>

Rendered element:

Reset Button Submit Button

Image as a Link

An Image as a Link is a <img> element nested within an <a> element

Images have four different sizes and must have a size associated with them. Images can be assigned by using the following CSS selectors:

  • .image-xs -> Extra Small Image
  • .image-sm -> Small Image
  • .image-md -> Medium Image
  • .image-lg -> Large Image

Sample code:

<a href="#"><img class="image-xs" src="images/matcha-cookies.png" alt="Illustration of a cat head"/></a>

Art Direction

Tamooki was imagined with first in mind, what sort of style did we want to go for. We knew the main color of the site would be blue. From there, we experimented with wavy patterns that resembled ocean waves but they were difficult to implement using the current tools. Although, from this process it was clear that the style we wanted was elements to have a round design. As we kept building onto this idea for Tamooki, we realized we wanted our brand to have a fun, relaxing, and colourful art direction. Our main inspiration at this point was Animal Crossing. The vibrant colors and tone was everything we were looking for. From there, we looked at similar sites such as Nookazon to draw further inspiration
